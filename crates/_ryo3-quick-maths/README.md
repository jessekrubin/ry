# `ryo3-quick-maths`

> **THIS IS A TEMPLATE LIBRARY**

---

A rust implementation of the "quick-maths" algorithm, as formally specified in
Shaq et al. (2017):

`2 plus 2 is 4, minus one that's 3, quick maths` [^1]

[^1]:
    [Big Shaq, M., et al. (2017). "Man's Not Hot".](https://youtu.be/3M_5oYU-IsU?t=64)
